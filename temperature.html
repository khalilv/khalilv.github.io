Automated Temperature and Humidity Monitoring System

  • Source
    • github.com/khalilv/temp-sense
  • Details

    I started this project in the summer of 2021, to help one of my friends, who is a working sous-chef, limit food waste in his restaurant. He mentioned how much food they throw out due to poor temperature monitoring, so I set out to build a lightweight module that would automatically take temperature and humidity readings in a given time interval, and export them to a server for analysis. This module could be placed on a food container in the restaurant to save busy employees from manually taking readings every 4 hours. Instead, they would be able to view the readings on a tablet in the kitchen and would get notified if the values exceed set thresholds. I built the prototype for this project and have done some initial validation, but I have not yet developed a product ready to pilot in his store.

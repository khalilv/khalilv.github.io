    I made this game in the summer of 2019. It was my first introduction to JavaScript, HTML and CSS. I started by watching a YouTube video on Intro to Game Development with JavaScript and by following instructions in an MDN document 2D breakout game using pure JavaScript. After some hours of work, to my surprise, I had a working brick breaker game that I could play on my browser. This was my first taste into the world of front-end development, and I was amazed that I could build something like that given the limited knowledge and experience I had.

    However, at this point, I wasn't fully satisfied with my work. Given I mostly just followed instructions in the video and in the document, I felt I needed to do a little more before calling a close to the project. So over the next few days I took some time to add extra features to personalize and improve my version of the game. Some features I added were, increased ball speed and smaller paddle width as the game progresses, pausing the game on new levels and on a new life, 5 level variations, and a more realistic rebounding of the ball off bricks. I also added custom styling to make the game look my own.

    The whole project took me just over a week to complete. It was a great learning experience for me to understand how JavaScript, HTML and CSS work together and I am glad I took the time to build the game. I use JavaScript, HTML and CSS almost everyday now and its always nice to look back and see where it all started.
